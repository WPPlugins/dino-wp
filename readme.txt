Ferramenta para visualização de notícias distribuídas pelo DINO - Visibilidade Online.

== Description ==

O DINO é uma ferramenta que  permite aos profissionais e não profissionais da área de comunicação, alavancar um release ou comunicado de imprensa para interagir com o seu público-alvo, com o objetivo de dar uma maior VISIBILIDADE ONLINE  a informação.

Esse plugin tem a função de divulgar as notícias distribuídas no DINO, trazendo conteúdo e mais visibilidade ao seu site.

O plugin possui a opção para configurar o css da notícia no seu site.

A opção de customização da notícia esta em constante desenvolvimento.

Se você tiver problemas com o plugin ou quer sugerir alguma modificação, por favor entre em contato com [DINO](mailto://contato@dino.com.br/).

== Installation ==

A instalação do plugin é bastante simples.

1. Extraia a pasta **dino-wp** para a pasta onde se encontra os seus plugins Ex.:**/wp-content/plugins/**
2. **Ative** o DINO WP no menu **Plugins** do Wordpress
3. Entre em contato com **contato@dino.com** para terminar a instalação
